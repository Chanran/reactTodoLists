# react+redux+react-router todoLists

## start

open a terminal
```
git clone https://github.com/Chanran/reactTodoLists
cd reactTodoLists
npm i -d
npm i -gd nodemon
npm start
```

open another terminal
```
npm run dev
```

open **[http://localhost:3001/login](http://localhost:3001/login)** in browser.
